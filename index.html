A PROPOS

À 19 ans, je suis actuellement en BTS Services informatiques aux organisations. J’ai acquis des compétences dans plusieurs langages web (HTML, CSS, JavaScript, PHP), ainsi qu'en gestion de bases de données MySQL. Je maîtrise également le développement d'applications lourdes et mobiles (C#, Python, Java, Flutter). Mon objectif est de poursuivre mes études avec un bachelor en développement informatique en alternance.

Compétences

HTML

Langage de balisage qui sert à structurer le contenu des pages web. Il permet d’organiser le texte, les images, les liens et autres éléments visibles sur un site. Il est la base de tout site web.

CSS

Langage de mise en forme qui permet de styliser les pages web. Il est utilisé pour définir les couleurs, polices, espacements, animations et assurer un design responsive, s’adaptant aux différents écrans.

JavaScript

Langage de programmation permettant d’ajouter de l’interactivité aux sites web. Il sert à créer des animations, gérer des événements (clics, formulaires), et à manipuler le contenu en temps réel.

PHP

Langage de programmation côté serveur permettant de générer des pages dynamiques. Il est principalement utilisé pour gérer les bases de données, créer des sessions utilisateurs et développer des applications web interactives

MySQL

Système de gestion de bases de données relationnelles permettant de stocker, organiser et récupérer des informations. Il est utilisé pour les sites web, applications métiers et solutions nécessitant une gestion efficace des données.

Wordpress

CMS (Content Management System) qui permet de créer et gérer facilement des sites web sans forcément coder. Utilisé pour des blogs, sites vitrines, e-commerce et même des plateformes complexes grâce à ses extensions.

C#

Langage de programmation orienté objet développé par Microsoft. Utilisé pour créer des applications de bureau sous Windows, des jeux vidéo (Unity), des logiciels métiers et des applications web avec ASP.NET.

Python

Langage polyvalent, simple et puissant. Utilisé dans de nombreux domaines comme le développement web (Django, Flask), l’intelligence artificielle, le big data, l’automatisation et le scripting.
